## ארכיטקטורת ניתוב

```
דביר / בקשה נכנסת
        ↓
  Orchestrator
  (קורא brain/router.md)
        ↓
┌──────────────────────────────────────────────────────┐
│ תפעולי: PM │ Finance │ Content │ Marketing │ Prod    │
│ אסטרטגי:  COO │ Tech Growth │ Business Growth       │
└──────────────────────────────────────────────────────┘
        ↓
   output/ ← כל תוצר נשמר כאן
        ↓
  דביר מאשר ← פעולות כספיות / שליחה / שינויים אסטרטגיים
```

## זרימת Growth → COO

```
Tech Growth  → שבועי  → דביר → אישור → COO מטמיע
Biz Growth   → דו-שבועי → דביר → אישור → COO מטמיע
```

---

## כללי Escalation משותפים לכל הסוכנים

- ליד חדש מסגמנט יעד → **התראה מיידית לדביר**
- הצעת מחיר מעל 15,000 ₪ → **אישור דביר לפני שליחה**
- הנחה מעל 10% → **אישור דביר**
- תשלום שלא הגיע +7 ימים → **דביר + Finance**
- פרויקט בעיכוב 48+ שעות → **דביר**
- שינוי ב-`memory/business_context.json` → **דביר תמיד**
- המלצה טכנולוגית / עסקית → **דביר מחליט, COO מבצע**
