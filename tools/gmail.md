```
1. בדוק ב-Supabase מתי נשלחה הצעת המחיר
2. אם עברו 3 ימים ללא תגובה → צור טיוטת פולואפ:
   נושא: "בהמשך להצעה — [שם הפרויקט]"
   גוף: [תבנית פולואפ מ-agents/client_relations.md]
3. שמור לוג ב-output/clients/[שם-לקוח]/followup_log.json
4. עדכן דביר: "טיוטת פולואפ מוכנה"
```

```

היי [שם],

בהמשך לשיחתנו — מצורפת הצעת המחיר ל[שם הפרויקט].
ההצעה בתוקף ל-14 יום. שמח לענות על כל שאלה.

דביר
Moon Studio
```

```

דביר
Moon Studio
```
