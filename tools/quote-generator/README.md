# Quote Generator — Moon Studio

> כלי Python שמייצר הצעות מחיר PDF ממותגות ב-RTL עברית.
> **Finance Agent משתמש בכלי זה לכל הצעת מחיר.**

---

## מה הכלי עושה

- מקבל פרמטרים טקסטואליים פשוטים
- מייצר PDF ממותג של Moon Studio עם:
  - כותרת שחורה עם לוגו לבן
  - פרטי הלקוח
  - פירוט שירותים
  - חישוב מע"מ 18% אוטומטי
  - סה"כ לפני ואחרי מע"מ
  - תנאי תשלום
- שומר PDF ב-`output/quotes/[quote_id]/`

---

## שימוש

### קלט (Input)

הסקריפט מצפה לקלט בפורמט הבא (מחרוזת טקסט):

```
שם לקוח: [שם מלא / חברה]
תאריך: [DD/MM/YYYY]
כותרת הפרויקט: [כותרת]
סכום: [מספר בלבד, ללא ₪]
פירוט: [תיאור השירותים — שורה אחת או כמה שורות]
הערות: [הערות אופציונליות]
```

### הרצה ישירה

```bash
cd tools/quote-generator
python3 scripts/generate_quote.py
```

הסקריפט יבקש את הנתונים interactively, או ניתן להזינם כ-stdin:

```bash
echo "שם לקוח: חברת ABC
תאריך: 15/03/2026
כותרת הפרויקט: סרט תדמית
סכום: 8000
פירוט: יום צילום מלא + משמרת עריכה" | python3 scripts/generate_quote.py
```

- קובץ PDF נשמר ב-`output/quotes/`
- שם קובץ: `quote_[תאריך]_[שם לקוח].pdf`

## כיצד Finance Agent משתמש בכלי

1. Finance Agent מקבל בקשה להצעת מחיר
2. מוציא נתונים מ-`memory/business_context.json` (מחירים עדכניים)
3. בונה את הקלט בפורמט הנכון
4. מריץ את הסקריפט
5. PDF נשמר ב-`output/quotes/[quote_id]/`
6. Finance Agent מעדכן את `quote.json` עם `pdf_path`
7. **דביר שולח את ה-PDF — הסוכן לא שולח לבד**

## הצעת מחיר — תקפות וכללים

- תוקף: **14 יום** מתאריך ההנפקה
- מע"מ: **18%** — מחושב אוטומטית
- תנאי תשלום: **50% מקדמה | 50% במסירה**
- הנחה מעל 10% → **לא לפרסם לפני אישור דביר**
- סכום מעל 15,000 ₪ → **לא לשלוח לפני אישור דביר**
