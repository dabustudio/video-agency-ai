## 1. פורמט דוח לדביר (Orchestrator)

```
📋 [כותרת קצרה — מקסימום 5 מילים]
- [עובדה 1]
- [עובדה 2]
- [עובדה 3 — רק אם יש]
⚡ נדרשת פעולה שלך: [מה בדיוק, פעולה אחת]
```

**כללים:**
- לא יותר מ-5 שורות סה"כ
- אם אין פעולה נדרשת — לא לכתוב את שורת ה-⚡
- לא לנפח — רק מה שדביר צריך לדעת

## 3. תגובה ראשונית ללקוח חדש

```
היי [שם],

תודה שפנית ל-Moon Studio!

שמחנו לשמוע על הפרויקט שלך. כדי שנוכל להגיש לך הצעה מדויקת, נשמח להבין קצת יותר:

• מה המטרה העיקרית של הסרטון?
• מה לוח הזמנים שלך?
• יש לך תקציב ראשוני בראש?

נחזור אליך תוך 24 שעות עם הצעה מותאמת.

Moon Studio
```

## 8. הצעת שיתוף פעולה (Cold Outreach)

```
שלום [שם],

אני דביר מ-Moon Studio — סטודיו הפקות וידאו שעובד עם מותגים ועסקים ב[תחום].

ראיתי את [חברה / מסעדה / מותג] ונראה לי שוידאו מקצועי יכול לעשות המון עבורכם —
בין אם זה [תוכן לסושיאל / פרסומת / סרט תדמית].

שמח לשלוח דוגמאות רלוונטיות. מתאים לשוחח בקצרה?

דביר
Moon Studio
```

**כללים ל-Cold Outreach:**
- מקסימום 5 שורות
- לא גנרי — לציין משהו ספציפי על העסק
- CTA אחד ברור
- לא לצרף PDF ראשוני — קודם לקבל תגובה
